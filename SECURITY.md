# Security Policy

## Reporting a Vulnerability

Please report any security issues to the maintainers via email at security@yourdomain.com.

## Security Practices

- Dependencies are regularly audited and updated.
- Backend uses Helmet for HTTP header security.
- CORS policies restrict access to trusted origins.
- Input validation is enforced on all API endpoints.
- Frontend uses React StrictMode to detect unsafe lifecycles.
- Sensitive files and environment variables are excluded from the repository.
- Static code analysis is performed using ESLint with security plugins.

## Updates

This document is updated with every security-related change.
